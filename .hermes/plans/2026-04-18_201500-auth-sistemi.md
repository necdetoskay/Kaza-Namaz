## 🔐 Auth Seçenekleri

### Seçenek 1: Basit PIN/Şifre (Önerilen)
```
• Kullanıcı uygulamaya PIN belirler
• Her açılışta PIN istenir
• Veriler cihazda kalır (güvenli)
• Kurulumu kolay
```
**Artıları:** Hızlı, basit, offline çalışır  
**Eksileri:** PIN unutulursa veri kaybı riski

---

### Seçenek 2: Firebase Auth
```
• Google, Email/Password, Phone
• Bulut senkronizasyonu
• Ücretsiz kota (50k/month)
```
**Artıları:** Çoklu cihaz, profesyonel  
**Eksileri:** Google hesabı gerekli, online bağımlılığı

---

### Seçenek 3: Biyometrik (Parmak İzi/Yüz)
```
• Cihazın parmak izi/yüz tanıma sensörü
• En hızlı çözüm
```
**Artıları:** En hızlı, kolay kullanım  
**Eksileri:** Her cihazda farklı destek

---

## 📋 Önerilen Mimari (Seçenek 1 + 3 Kombinasyonu)

### Katman 1: PIN Koruması
```
• 4-6 haneli PIN kodu
• 3 yanlış deneme = 30 dakika bekleme
• PIN hash'lenerek saklanır (SHA-256)
```

### Katman 2: Biyometrik (Opsiyonel)
```
• Cihaz destekliyorsa parmak izi/yüz
• PIN'e alternatif
```

### Katman 3: Veri Şifreleme
```
• SQLite veritabanı şifreleme
• SQLCipher ile
• PIN hash'i encryption key olarak
```

## 🔧 Teknik Detaylar

### PIN Saklama
```typescript
// PIN hash'leme
const hash = await crypto.subtle.digest('SHA-256', pinEncoder.encode(pin));
const hashString = Array.from(new Uint8Array(hash))
  .map(b => b.toString(16).padStart(2, '0'))
  .join('');

// Saklama (SQLite)
// NOT: PIN asla plaintext saklanmaz!
```

### Encryption Key Üretimi
```typescript
// PIN'den encryption key türetme
const keyMaterial = await crypto.subtle.importKey(
  'raw', pinEncoder.encode(pin), 'PBKDF2', false, ['deriveBits']
);
const key = await crypto.subtle.deriveBits(
  { name: 'PBKDF2', salt, iterations: 100000, hash: 'SHA-256' },
  keyMaterial, 256
);
```

### SQLite Şifreleme
```typescript
// SQLCipher ile
sqliteDb = await sqlite.open({
  name: DB_NAME,
  location: 'default',
  encrypted: true,
  mode: 'encryption'
});
```

## ⚠️ Riskler

| Risk | Çözüm |
|------|-------|
| PIN unutulursa | Kurtarma kodu (backup) |
| Biyometrik başarısız | PIN fallback zorunlu |
| Cihaz değişikliği | Export/import ile veri taşıma |
