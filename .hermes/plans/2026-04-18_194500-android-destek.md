# Plan: Android Desteği Ekleme (Capacitor)

## 🎯 Hedef
Kaza-Namaz uygulamasını Android APK olarak build edilebilir hale getirmek.

---

## 📊 Seçenek Karşılaştırması

| Özellik | Capacitor (Önerilen) | PWA |
|---------|---------------------|-----|
| **Gerçek APK** | ✅ Evet | ❌ Hayır |
| **Play Store** | ✅ Evet | ⚠️ Manuel |
| **Push Bildirimi** | ✅ Evet | ⚠️ Service Worker |
| **Kurulum** | ✅ Evet | ✅ Evet |
| **İş Süresi** | ~30-60 dk | ~15-30 dk |
| **Karmaşıklık** | Orta | Düşük |

**Karar:** Capacitor (gerçek APK + Play Store potansiyeli)

---

## 🔧 Adımlar

### 1. Capacitor Kurulumu
```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
npx cap init "Kaza Namaz" "com.kazanamaz.app" --web-dir=dist
npm run build
npx cap add android
npx cap sync android
```

### 2. Android Studio'da Build
- `android/` klasörünü Android Studio'da aç
- Build → APK generate et

### 3. Olası Ek Yapılandırmalar
- `android/app/src/main/res/` → Icon ekleme
- `AndroidManifest.xml` → Permissions (internet, vibration)
- Splash screen (opsiyonel)

---

## ✅ Doğrulama
1. `npm run build` → dist/ oluşur
2. `npx cap sync android` → android/ güncellenir
3. Android Studio'da APK build edilir
4. Emülatör veya gerçek cihazda test edilir

---

## ⚠️ Notlar
- İlk build uzun sürebilir (Android Studio SDK download)
- LocalStorage veri korunur (aynı web app)
- PWA değil, gerçek native APK üretilir
